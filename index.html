HTML-sivun perusrakenne

Jokainen HTML-sivu koostuu samoista perusosista. Alla on esimerkki minimisivusta.

index.html 
<!DOCTYPE html>
<html lang="fi">
  <head>
    <meta charset="UTF-8" />
    <title>Sivun otsikko</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>

    <h1>Tervetuloa!</h1>
    <p>Tämä on tekstikappale.</p>

  </body>
</html>

Tärkeimmät HTML-tagit

Tagit ovat HTML:n rakennuspalikoita. Jokainen tagi aloitetaan <tagname> ja lopetetaan </tagname>.

<html>
HTML-elementti

Sivun juurielementti. Kaikki muu sisältö on tämän sisällä. lang-attribuutti kertoo kielen.

<html lang="fi"> ... </html>
<head>
Head-osio

Sisältää metatiedot sivusta: otsikon, CSS-linkit ja merkistön. Ei näy käyttäjälle suoraan.

<head> <title>Otsikko</title> </head>
<body>
Body-osio

Kaikki sivun näkyvä sisältö: teksti, kuvat, linkit ja muut elementit sijoitetaan tänne.

<body> ... näkyvä sisältö ... </body>
<h1> – <h6>
Otsikkotagit

Määrittävät otsikkotasot. <h1> on tärkein, <h6> pienin. Tärkeä SEO:lle.

<h1>Pääotsikko</h1>
<h2>Alaotsikko</h2>
<p>
Kappale

Tekstikappale. Selain lisää automaattisesti väliä ennen ja jälkeen kappaleen.

<p>Tämä on tekstikappale.</p>
<a>
Linkki (Anchor)

href-attribuutti määrittää kohteen. target="_blank" avaa uuteen välilehteen.

<a href="https://google.fi" target="_blank">Google</a>
<img>
Kuva

Itsesuljettava tagi. src on kuvan polku, alt vaihtoehtoinen teksti saavutettavuutta varten.

<img src="kuva.jpg" alt="Kuvaus" />
<br>
Rivinvaihto

Pakottaa rivinvaihdon ilman uutta kappaletta. Itsesuljettava tagi — ei lopetustaggia tarvita.

Rivi yksi<br />
Rivi kaksi
<div>
Lohko-elementti

Yleinen säiliö ryhmittelyyn. Ei itsessään näytä mitään — käytetään CSS:n ja Bootstrapin kanssa asetteluun.

<div class="container"> ... </div>

CSS-muotoilu

CSS (Cascading Style Sheets) määrittää, miltä HTML-elementit näyttävät. Muotoilua voidaan lisätä kolmella tavalla.

01
Ulkoinen tiedosto

Suositeltavin tapa. CSS kirjoitetaan erilliseen .css-tiedostoon ja linkitetään HTML:ään.

<link rel="stylesheet" href="style.css" />
02
Style-tagi

CSS kirjoitetaan suoraan HTML-tiedostoon <style>-tagin sisään <head>-osiossa.

<style> body { color: red; } </style>
03
Inline-tyyli

Tyyli lisätään suoraan elementin style-attribuuttiin. Ei suositella ylläpidettävyyden vuoksi.

<p style="color: blue;">Teksti</p>
style.css — esimerkki 
/* Valitsin: kaikki p-tagit */
p {
  color: #333333;
  font-size: 16px;
  line-height: 1.6;
}

/* Luokkavalitsin: .card */
.card {
  background: white;
  border-radius: 8px;
  padding: 20px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

/* ID-valitsin: #hero */
#hero {
  background-color: #0d1117;
  color: white;
}

Bootstrap-kirjasto

Bootstrap on suosittu CSS-kirjasto, joka tarjoaa valmiita komponentteja ja responsiivisen grid-järjestelmän. Linkitetään sivulle CDN:n kautta.

Grid-järjestelmä

Bootstrap jakaa rivin 12 sarakkeeseen. Eri näyttökokoihin voidaan määrittää eri leveydet.

<div class="row">
  <div class="col-md-6">Vasen</div>
  <div class="col-md-6">Oikea</div>
</div>
Valmiit komponentit

Navbar, Card, Button, Modal, Alert — kaikki valmiina, tarvitset vain oikean class-nimen.

<button class="btn btn-primary">
  Klikkaa tästä
</button>
